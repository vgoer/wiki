## python打包

> python 打包为二进制程序





### 1. 安装

>  PyInstaller

```shell
pip install pyinstaller


```





### 2. 打包

> 打包

```shell
pyinstaller your_script.py
```

> 打包完成后，在 `dist` 目录下会生成一个可执行文件，可以直接运行该文件来执行你的 Python 程序。







### 3. 指定包含的模块

> 添加模块，打包进去 `requirements.txt` 文件
>
> 直接把当前的模块写入到 文件中。

```shell
pip freeze > requirements.txt
```
