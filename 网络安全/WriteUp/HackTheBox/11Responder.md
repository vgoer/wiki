```bash

# CentOS/RHEL
sudo dnf install samba samba-winbind
```


#### 配置 Samba
```ini:/etc/samba/smb.conf
[global]
    workgroup = WORKGROUP
    security = ads
    realm = DOMAIN.COM
    winbind use default domain = yes
    winbind enum users = yes
    winbind enum groups = yes
    template shell = /bin/bash
```
