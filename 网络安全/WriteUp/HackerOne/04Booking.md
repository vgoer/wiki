### 1. 信息收集

> 子域名收集

```shell
subfinder -d booking.com -o all_subdomains.txt

# 存活的域名
sudo apt install httpx-toolkit
cat all_subdomains.txt | httpx-toolkit > live_subdomains.txt

```
