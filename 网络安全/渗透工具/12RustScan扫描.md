## RustScan扫描

> 现代端口扫描器。 [github](https://github.com/RustScan/RustScan)







### 1. 安装

```shell
# 安装snap
sudo apt install snapd
sudo snap install snapd
sudo systemctl restart snapd
sudo systemctl enable snapd

# 安装rustscan
snap install rustscan 

# 帮助手册
rustscan -h
```
