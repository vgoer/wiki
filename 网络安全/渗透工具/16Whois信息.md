## Whois信息

> whios是个标准的互联网协议,可用于收集网络注册,注册域名,IP地址和自治系统的信息,whios数据库记录有该域名的DNS服务器信息和注册人的联系信息。
>
> [kali](https://www.kali.org/tools/whois/)  [blog](https://blog.csdn.net/u011781521/article/details/69225784)







### 1. 安装

```shell
# kali自带
sudo apt install whois

# whois信息
whois target.com
```
