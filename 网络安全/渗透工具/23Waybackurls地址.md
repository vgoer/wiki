## Waybackurls地址

> Waybackurls地址: 获取 Wayback Machine 所知道的某个域名的所有 URL
>
> [github](https://github.com/tomnomnom/waybackurls)









### 1. 安装

```shell
go install github.com/tomnomnom/waybackurls@latest

# 使用
cat domains.txt | waybackurls > urls
```
