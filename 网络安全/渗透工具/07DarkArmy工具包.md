## DarkArmy工具包

> DARKARMY 黑客工具包 - 一个渗透测试框架。[github](https://github.com/D4RK-4RMY/DARKARMY)







### 1. 安装



此工具必须以 ROOT 身份运行！！！

```
git clone https://github.com/D4RK-4RMY/DARKARMY.git
cd DARKARMY
chmod +x install.sh
./install.sh
```

就是这样。您可以通过输入**DARKARMY来执行该工具**
