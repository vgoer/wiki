> v-on: 事件
>
> `createApp({})` 参数六 `methods属性: 函数方法`
>
> v-if 条件判断
>
> `:style=“{样式名: vue数据}”`
>
> `:class="{类名: 布尔值}"`

```html
<form v-on:submit.prevent="add">
    <input type="text" placeholder="内容" v-model="content">
    <button type="submit"> add </button>
</form>
<script>
methods:{
    add(){
        this.walls.push({
            id: this.walls.length +1,
            name: this.content,
            image: './images/wall4.jpg',
            purchased:false,
        })
        this.content = ''
    }
}
</script>
```

> $特殊字符，向全局暴露自己的位置
>
> `$emit`暴露给父组件使用

```js
// form功能拆 AppForm.js
export default{
    template: /*html*/`
    <form v-on:submit.prevent="add">
        <input type="text" placeholder="内容" v-model="content">
        <button type="submit"> add </button>
    </form>
    `,
    data(){
        return{
            content:'',
        }
    },
    methods:{
        add(){
            // 暴露方法，传递参数
            this.$emit('add',this.content)
            this.content = ''
        }
    },
}
```
