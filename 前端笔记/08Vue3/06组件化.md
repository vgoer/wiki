## 组件化

> 将`app`分割为 多个组件   更好维护等
>
> `component('组件名称',{})`   注册组件

```vue
<div id="app">
    // 新组件
    <hivue></hivue>
</div>
<script src="https://unpkg.com/vue@next"></script>
<script>
    const app = Vue.createApp({
        data(){
            return{
                go:'aaa'
            }
        },
        methods:{

        },

    })

    // 新组件  component 注册组件   template 模板
    app.component('hivue',{
        data(){
            return{
                msg:'vue 组件'
            }
        },
        methods:{
            tempF(){
                console.log('1111')
            }
        },
        template:`
                <h1 @click="tempF">{{msg}}</h1>
            `
    })
    app.mount('#app')
</script>
```

### 作业

> 组件化开发实现一个水果列表 

```vue
<div id="app">
    <fruit-list></fruit-list>
    <fruit-list></fruit-list>
    <fruit-list></fruit-list>
    <fruit-list></fruit-list>
</div>
<script src="https://unpkg.com/vue@next"></script>
<script>
    const app = Vue.createApp()

	// 组件的嵌套
    app.component('fruit-list',{
        template:`
                <h2>水果列表</h2>
                <btn-list></btn-list>
            `
    })  
	
    app.component('btn-list',{
        data(){
            return{
                num:0,
            }
        },
        methods:{
            add(){
                this.num++
            },
            rem(){
                this.num--
            }
        },
        template:`
                <button @click="add">+</button>
                <span>{{num}}</span>
                <button @click=rem>-</button>
            `
    })

    app.mount('#app')

</script>
```



### 根组件

> app 是 实例对象

```vue
const = rootComp = app.mount('#app');

app 是 实例对象 
this === 根组件   true
根组件的 模板就是 绑定的 #app
```



### 局部组件

> `component` 注册的是全局组件
>
> `components` 注册的是局部组件  ==常用==

```vue

<div id="app">
    <frust-list></frust-list>
</div>
<script src="https://unpkg.com/vue@next"></script>
<script>

    let frustList = {
        template:`
                <h1>水果列表</h1>
            `
    }
    const app = Vue.createApp({

        // 注册局部组件
        components:{
            'frust-list': frustList
        }
    })
    app.mount('#app')

</script>
```
