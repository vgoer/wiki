## 列表渲染

> `v-for` 渲染列表  ==常用==

```vue
<div id="app">
    <ul>
        // v-for 渲染
        <li v-for="item in list">{{item}} </li>
    </ul>
</div>
<script src="https://unpkg.com/vue@next"></script>

<script>
    Vue.createApp({
        data(){
            return {
                list:['go','java','javascript','c']
            }
        }
    }).mount('#app')

</script>
```

### 双向数据绑定

> 数据和输入的同时修改

```vue
<div id="app">
    <h1>{{msg}}</h1>
    // v-model 双向绑定数据
    <input type="text" v-model="msg">
</div>
<script src="https://unpkg.com/vue@next"></script>

<script>
    Vue.createApp({
        data(){
            return {
                msg:'vue'
            }
        }
    }).mount('#app')
</script>
```

> v-model常用修饰符  ==常用==

```vue
v-model        => 默认是字符类型

v-model.number => 输入的是数字类型的

v-model.lazy   => 失去焦点绑定

v-model.trim  => 去除左右空格再绑定
```



### 作业

> 做一个购物车的简单版	

```vue
<div id="app">
    <h2>购物车</h2>

    <form @submit.prevent="reDate">
        名称: <input type="text" v-model="name">
        单价: <input type="text" v-model="price">
        <button>提交</button>
    </form>

    <ul>
        <li v-for="item,index in list">
            名称:{{item.name}};
            单价:{{item.price}};
            数量: 
            <button @click="rem(index)">-</button>
            <span>{{item.count}}</span>
            <button @click="add(index)">+</button>
        </li>
    </ul>

    <h3>总价:{{totalPrice}}</h3>
</div>
<script src="https://unpkg.com/vue@next"></script>
<script>
    Vue.createApp({
        data(){
            return {
                name:'',
                price:'',
                list:[]
            }
        },
        // 计算属性
        computed:{
            totalPrice(){
                let sum = 0;
                this.list.forEach(item => {
                    sum += (item.price * item.count)
                })        
                return sum                
            }
        },

        methods:{
            reDate(){
                this.list.push({
                    'name':this.name,
                    'price': this.price,
                    'count': 1,
                })
            },
            add(i){
                this.list[i].count++
            },
            rem(i){
                this.list[i].count--
                console.log(this.list[i].count)
                // 如果为0删除自己
                if( this.list[i].count <= 0){
                    this.list.splice(i,1)
                }
            }
        }
    }).mount('#app');
</script>
```
