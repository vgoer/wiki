## 自定义对象

> 先前我们学的都是内置对象，现在我们自定义对象

```js
// 自定义对象属性 
function Dog(name,age,coatColor){
    // 参数传递进来后赋值给属性
    this.name = name;
    this.age = age;
    this.coctColor = coatColor;   
    
    // 定义对象方法
    this.run=function(){
        this.age--;
    }
    this.monny = function(a){
        this.coctColor == a
    }    
    
}
// 这样我们就定义好了一个对象属性和方法
```



> 要使用自定义对象，必须`实例化new`
>
> 才能使用对象内的属性和方法

```js
var xiaozhi = new Dog('小智','4','白色');

console.log(xiaozhi.name);
xiaozhi.monny('黑色');

```
