### style和class绑定

> 实现动态绑定，常用

```html
// v-bind:class 加类名
<div id="app">
<h1 v-bind:class="{active:isactive,green:isgreen"}>{{msg}}</h1>
</div> //添加class属性


<script src="../script/vue.js"></script>
<script>
    new Vue({
        el:'#app',
        data:{
            msg:'hello Vue!',
            isactive:false,  //赋值
            isgreen:true,
        }
    });
</script>
<style>
    .active{
        display: none;
    }
    .green{
        color: green;
    }
</style>
//还可以是数组(数组是直接是类名)
v-bind:class = "['active','green']"

//三元运算符
 <h1 v-bind:class="[isactive ? 'active':'', isgreen ? 'green':'']">{{msg}}</h1>
```



> 内联样式

```html
// v-bind:style  绑定内联样式
<div v-bind:style="styleDiv">11111</div>
data:{
	styleDiv:{
		color:'blue',
		fontSize:'30px',
		....
	}
}
```
