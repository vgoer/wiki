### 1. 搭建

> 为了方便环境，我直接使用docker了 [doc](https://catchadmin.com/docs/3.0/catchadmin/install)

```shell
git clone https://github.com/jaguarjack/catch-admin 

# 创建test网络
docker network create test

# 启动容器
docker-compose up -d
```
