## wd-dist

> 无道词典：有道词典的命令行版本，支持英汉互查和在线查询。 [github]()





### 1. 安装

```shell
# 安装python
sudo pacman -S python3


git clone https://github.com/chestnutheng/wudao-dict
cd ./wudao-dict/wudao-dict
sudo bash setup.sh #或者sudo ./setup.sh
```





### 2. 使用

```shell
wd master
wd 傻逼
```
