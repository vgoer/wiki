### 2. 参数校验

>  很重要

```php
# 实际情况前后端都应该加参数校验
    public function store()
    {
        //  validate 参数校验
        request()->validate([
            // required 必填   max:最大字符
            'username' => ['required','max:20']
        ]);
        
        return redirect('/');
    }

# 显示报错信息
    @error('username')
        <p>
        	// 全局报错对象
            {{$errors->first('username')}}
        </p>
    @enderror

```

old() 保存了上次提交的数据
