## 接口

> go接口： 新的==类型定义==，把具有共性的方法定义到一起

```go
// 实现 手机和电脑usb读写的接口
type USBer interface {
	read()
	write()
}

type Computer struct {
	name string
}

func (m Computer) read() {
	fmt.Printf("我 %v 在读\n", m.name)
}
func (m Computer) write() {
	fmt.Printf("我 %v 在写\n", m.name)
}

func main() {

	hp := Computer{name: "惠普"}

	hp.read()
	hp.write()
}
```

> 实现接口定义的所有方法才行 不然报错



### 1. 接口的接受者类型和指针类型接受者

> 值类型不会影响外面的值
>
> 指针类型修改外面的值

```go
type Peter interface {
	eat()
}

type Person struct {
	name string
}

func (per Person) eat() {
	per.name = "phper"

}

func (per *Person) run() {
	per.name = "runer"

	fmt.Printf("per.name: %v\n", &per.name)

}

func main() {
	tom := Person{name: "tom"}

	fmt.Printf("tom: %v\n", tom)

	tom.eat()
	fmt.Printf("tom: %v\n", tom)

	tom.run()
	fmt.Printf("tom: %v\n", tom) // 外面被修改了 tom: {runer}
}
```



### 2. 接口和类型的关系

> 1. 一个类型多个接口
> 2. 多个类型可以实现同一个接口(多态)

```go
// 一个类型实现多个接口
Player 接口播放音乐 Videoer接口实现播放视频 手机Mobile 实现两个接口

type Player interface {
	playMusic()
}

type Video interface {
	playVideo()
}

type Mobile struct {
}

func (m Mobile) playMusic() {
	fmt.Println("播放音乐")
}

func (m Mobile) playVideo() {
	fmt.Println("播放视频")
}

func main() {

	mm := Mobile{}
	mm.playMusic()
	mm.playVideo()
}
```

```go
// 多个类型实现同一个接口
type Pet interface {
	eat()
}

type Cat struct {
}

type Dog struct {
}

func (c Cat) eat() {
	fmt.Println("cat eat...")
}
func (d Dog) eat() {
	fmt.Println("cat eat...")
}

func main() {

	var pet Pet
	// 多态 同一方法 内容不同
	pet = Dog{}
	pet.eat()

	pet = Cat{}
	pet.eat()
}
```



### 3.接口嵌套

> 接口嵌套实现新的接口

```go
type Flyer interface {
	fly()
}

type Swimer interface {
	swim()
}

// 新接口包含两个接口
type FlyFish interface {
	Flyer
	Swimer
}

type Fish struct {
}

func (fish Fish) fly() {
	fmt.Println("flying ....")
}

func (fish Fish) swim() {
	fmt.Println("swimming ...")
}
func main() {

	// 声明接口类型
	var ff FlyFish

	ff = Fish{}
	ff.fly()
	ff.swim()
}
```



### 4.OCP 设计原则

> 面向对象的可复用的第一块基石`开闭原则`ocp
>
> 对扩展是开放的， 修改是关闭的

```go
// 可以扩展 不能修改
// 动物接口
type Pet interface {
	eat()
	sleep()
}

type Cat struct {
}

// cat实现接口
func (c Cat) eat() {
	fmt.Println("cat eat...")
}
func (c Cat) sleep() {
	fmt.Println("cat sleep...")
}

type Dog struct {
}

// dog实现接口
func (d Dog) eat() {
	fmt.Println("dog eat...")
}
func (d Dog) sleep() {
	fmt.Println("cat sleep...")
}

// 人养宠物
type Person struct {
}

// 也可以养猫和狗
func (per Person) care(pet Pet) {
	pet.eat()
	pet.sleep()
}

func main() {
	dog := Dog{}
	cat := Cat{}

	per := Person{}

	per.care(dog)
	per.care(cat)
}
```

### 6.继承

> 结构体实现继承

```go
// 动物结构体有方法， 猫和狗没有方法，继承动物的方法
type Pet struct {
	name string
	age  int
}

func (pet Pet) eat() {
	fmt.Println("eat ...")
}
func (pet Pet) sleep() {
	fmt.Println("sleep ...")
}

type Cat struct {
	Pet
	color string
}

type Dog struct {
	Pet
	color string
}

func main() {
	// 声明时注意写法
	cat := Cat{
		Pet{name: "goer", age: 1},
		"白色",
	}

	// 调用方法
	cat.eat()

	fmt.Printf("cat: %v\n", cat)

	dog := Dog{
		Pet{name: "java", age: 3},
		"黑色",
	}
	dog.sleep()
	fmt.Printf("dog: %v\n", dog)
}
```



### 7.构造函数

> 初始化对象，go 使用函数模拟构造函数功能

```go
// 构造函数
type Person struct {
	name string
	age  int
}

// 函数实现构造函数功能   两个返回值
func newPerson(name string, age int) (*Person, error) {

	if name == "" {
		return nil, fmt.Errorf("name不能为空")
	}
	if age < 0 {
		return nil, fmt.Errorf("age 不能小于0")
	}
	// 取指针的地址
	return &Person{name: name, age: age}, nil
}

func main() {

	tom, err := newPerson("tom", 10)

	if err == nil {
		// 取地址的值
		fmt.Printf("tom: %v\n", *tom)
	} else {
		fmt.Printf("err: %v\n", err)
	}
}
```
