## 常用命令

> go 比较常用的命令 
>
> go ==run== ==get== ==build==



```shell
1.  go env   # 环境变量

2. go run main.go  #直接运行源码程序

3. go build # 编译包和依赖  
go build -o heiheihei.exe # -o 指定文件名

4. go get + package  # 下载包

5. go test #测试

6. go tool # go提供工具

7. go fmt #格式化

```

> 初始化

```shell
go mod init 项目名称  #初始化

```
