## 镜像操作

> 操作一系列镜像的命令

### 1. 仓库

> 一些列镜像存放的地方

```shell

# 官方仓库

https://hub.docker.com/

# 推荐

https://hub.daocloud.io/

```

### 2.操作

> 简单操作

```shell

1. 拉取镜像到本地

docker pull 镜像名称[:tag版本]

docker pull tomcat  #latest 默认最新

2. 查看仓库

docker images

3. 删除仓库

docker rmi ***12da # 前面三个字符即可

4. 查看容器运行

docker ps

5. 删除容器

docker rm *** # 前几个字符即可

6. 镜像导出

docker save -o/路径+文件名 镜像id  # 前几个字符即可

7. 导入

docker load -i 镜像文件

8.修改镜像名称和版本

docker tag 镜像id  新名称:版本

```
