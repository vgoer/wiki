## 搭建服务

> 搭建和套服务，用上面学习的知识

```yaml
// docker.compose.yml

version: '3.8'
servers: 
	db: 
		image: mariadb  # mariadb 数据库
		environment： 
			MYSQL_ROOT_PASSWORD:'root'
			MYSQL_DATABASE:'app'
			MYSQL_USER:'app'
			MYSQL_PASWORD:'app123'
		volumes:
			-db:/var/lib/mysql
	php: 
		image: bitnami/php-fpm
		volumes:
			
valumes:
	db:
	driver:local
```
