## wayfire

>模块化和可扩展的 Wayland 合成器 [way](https://wayfire.org/)



## 1. 安装

> github[way](https://github.com/WayfireWM/wayfire)

```shell
# 添加cn源
[archlinuxcn]
https://mirrors.tuna.tsinghua.edu.cn/help/archlinuxcn/$arch
sudo pacman -S archlinuxcn-keyring  # 钥匙

yay -S wayfire  # 其他发行版
wayfire -v  # 查看版本
```
